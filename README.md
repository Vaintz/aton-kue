# Aton Kue - Website Portofolio UMKM

Website portofolio untuk UMKM **Aton Kue** yang dibuat sebagai proyek mata kuliah **Technopreneurship**.

Website ini bertujuan untuk membantu promosi digital UMKM melalui katalog produk, informasi usaha, galeri, testimoni pelanggan, serta kontak yang terintegrasi dengan WhatsApp.

## Cara Menjalankan

1. Download atau clone project.
2. Buka folder menggunakan Visual Studio Code.
3. Jalankan menggunakan Live Server.
4. Website akan terbuka di browser.

## Tujuan

Website ini dikembangkan sebagai media promosi digital untuk UMKM Aton Kue agar dapat:

- Memperkenalkan profil usaha.
- Menampilkan katalog produk.
- Mempermudah pelanggan menghubungi penjual melalui WhatsApp.
- Meningkatkan branding dan citra usaha.
